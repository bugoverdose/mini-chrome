Mini Chr me

Mini Chrome is a simple and light-weight web browser, developed by adding User Interface on top of Chronium.

Download

After downloading, unzip the .zip file.

Inside the folder, find and run the .app or .exe file to run the application.

About

I started this project because I wanted to have a little taste of what building a multi-process application is like.

Compared to building a website or a backend application, I felt more powerful as I gained so much control over the device.

I stopped working on it because I've already learned enough from this project. I could add more UI and functionalities, but that would just be copy/pasting the code I've already wrote.

Also, we all know that no one's actually going to be using this app.
